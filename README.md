# heuristic-slide-puzzle

 ## 환경 구축

 ### 프로젝트 생성

 1. Visual Studio 2022 실행
 2. 빈프로젝트로 프로젝트 생성

 ### 레포지토리 클론

 프로젝트 폴더 루트 경로에 레포지토리 클론
 ```
 git clone https://github.com/Woolegend/heuristic-slide-puzzle.git
 ```

git이 안 깔려있다면 .zip으로 코드를 다운받은 뒤 프로젝트 폴더에 압축 해제
